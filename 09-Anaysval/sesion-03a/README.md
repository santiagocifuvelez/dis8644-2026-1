# sesion-03a

Hoy seguimos trabajando con el chip 555 en un circuito astable, que al principio tenía un LED con una resistencia. Después cambiamos el LED y la resistencia por un capacitor y un parlante. Para hacer las conexiones usamos cables caimán.

### Conceptos:

- LDR = fotoresistor
- Circuito astable: circuito sin estabilidad
- Frecuencia: cada cuánto ocurre algo o cuántas cosas pasan en un determinado tiempo
- Período: se mide en tiempo y es el tiempo que hay entre que se repite el ciclo de una onda

Luego de revisar varias veces no sabíamos qué más cambiar por lo que revisamos los resultados de algunos compañeros e hicimos algunos cambios.
